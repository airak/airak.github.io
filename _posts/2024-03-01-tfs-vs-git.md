O TFVC é o sistema de controle de versão centralizado criado para o TFS.

## TFS (Team Foundation Server)

O TFS ou Azure DevOps Server é um conjunto de ferramentas colaborativas de desenvolvimento de software, o seu controle de versionamento de código pode ser realizado tanto pelo GIT como pelo TFVC.

## Diferenças entre o GIT e o TFVC

O Git é um sistema de versionamento descentralizados enquanto o TFVC é centralizado 

| TFVC  | GIT |
| ------------- | ------------- |
| Centralizado  | Descentralizado  |
| Content Cell  | Content Cell  |
